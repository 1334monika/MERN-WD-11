Understanding CSS Colors

In CSS, colors can be applied to elements to change their text color, background color, borders, and more. There are several ways to define a color.

1. Color Keywords (Named Colors)

CSS has a list of predefined color names that you can use directly. There are over 140 named colors, including common ones like red, blue, green, orange, and transparent. 

.element {
  color: dodgerblue;
  background-color: lightgray;
}

2. RGB and RGBA Colors

The RGB (Red, Green, Blue) model defines a color by mixing red, green, and blue light. Each value can be an integer between 0 and 255.

The RGBA model adds an "alpha" channel, which controls the opacity of the color. The alpha value is a number between 0.0 (fully transparent) and 1.0 (fully opaque).

/* RGB - Solid Tomato Color */
.solid-color {
  background-color: rgb(255, 99, 71);
}

/* RGBA - 50% Transparent Tomato Color */
.transparent-color {
  background-color: rgba(255, 99, 71, 0.5);
}

3. Hexadecimal (HEX) Colors

HEX colors are a very common way to specify colors. They are a hexadecimal representation of the RGB values. The format starts with a # followed by six characters (RRGGBB).

A shorthand 3-digit version (#RGB) can be used if each pair of characters is the same (e.g., #FF0000 can be written as #F00).

/* 6-digit HEX - Lime Green */
.full-hex {
  color: #32CD32;
}

/* 3-digit shorthand HEX - A shade of magenta */
.shorthand-hex {
  background-color: #F0C; /* Equivalent to #FF00CC */
}

4. HSL and HSLA Colors

The HSL (Hue, Saturation, Lightness) model is often more intuitive for humans to work with.

  • Hue: A degree on the color wheel (0 to 360). 0 is red, 120 is green, 240 is blue.
  • Saturation: A percentage value. 0% is a shade of gray, and 100% is the full color.
  • Lightness: A percentage value. 0% is black, 50% is the normal color, and 100% is white.

Like RGBA, HSLA adds an alpha channel for opacity.

/* HSL - A pure, vibrant blue */
.vibrant-blue {
  background-color: hsl(240, 100%, 50%);
}

/* HSLA - The same blue, but 50% transparent */
.transparent-blue {
  background-color: hsla(240, 100%, 50%, 0.5);
}
