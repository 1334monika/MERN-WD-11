Introduction to CSS

Welcome to CSS! This guide will introduce you to the basic concepts of what CSS is and how to use it.

What is CSS?

CSS stands for Cascading Style Sheets. It is the language we use to style an HTML document. CSS describes how HTML elements should be displayed on screen, on paper, or in other media.

Think of HTML as the skeleton of a webpage (the structure and content), and CSS as the clothes, colors, and overall appearance (the presentation).

Why Use CSS? The Separation of Concerns

The most important reason to use CSS is to separate the content and structure (HTML) from the presentation (CSS). This makes your code much easier to manage.

  • Easier Maintenance: You can change the entire look of a website by editing just one CSS file.
  • Faster Load Times: One CSS file can be used for multiple pages, and the browser can cache it, speeding up page loads.
  • Cleaner HTML: Your HTML files become smaller and more readable without all the styling information cluttered inside them.

How to Add CSS to a Page

There are three ways to include CSS in an HTML file.

1. External CSS (Recommended)

This is the best method. You write your CSS rules in a separate file with a .css extension and link to it from your HTML file using a <link> tag in the <head> section. 

<!DOCTYPE html>
<html>
<head>
  <title>My Website</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  ...
</body>
</html>

2. Internal CSS

You can place CSS rules directly inside the HTML file by putting them within a <style> tag in the <head> section. This is useful for single pages or for styles that only apply to one specific page. 

<!DOCTYPE html>
<html>
<head>
  <title>My Page</title>
  <style>
    body {
      background-color: lightblue;
    }
    h1 {
      color: navy;
    }
  </style>
</head>
<body>
  ...
</body>
</html>

3. Inline CSS

You can apply styles directly to a single HTML element using the style attribute. This method is generally discouraged because it mixes content and presentation, making the code harder to maintain. 

<h1 style="color: blue; text-align: center;">A Blue, Centered Heading</h1>
<p style="color: red;">A red paragraph.</p>

Basic CSS Syntax

A CSS rule consists of a selector and a declaration block.

Image credit: w3schools.com

  • The selector points to the HTML element you want to style (e.g., p).
  • The declaration block contains one or more declarations separated by semicolons.
  • Each declaration includes a CSS property name and a value, separated by a colon (e.g., color: red;). 
/* This entire block is a CSS rule */

/* 'p' is the selector */
p {
  /* 'color: red;' is a declaration */
  color: red; /* 'color' is the property, 'red' is the value */
  font-size: 16px;
}
